This repository is a personal NixOS/nix-darwin/Home Manager configuration built around a modular assembly model. Each output is composed from the smallest useful set of reusable pieces — hosts, system profiles, home profiles, and explicit overlay groups — with heavyweight features added deliberately rather than arriving through hidden coupling.

The flake targets eight outputs across three platforms: NixOS (x86\_64 and ARM), nix-darwin (Apple Silicon), and standalone Home Manager (Fedora, Jetson Orin Nano). All outputs share a common module layer and a repo-local `my.*` options namespace for threading values across NixOS and Home Manager boundaries.

## How the repo is structured

The configuration is split into four layers:

- **`hosts/`** — machine-specific hardware, storage, networking, users, and service-local values
- **`modules/system/`** — reusable NixOS modules: TTY baseline, Sway compositor, gaming, containers, services
- **`modules/home/`** — reusable Home Manager modules: CLI tools, desktop packages, dotfiles, shell helpers
- **`modules/shared/`** — repo-local `my.*` options that cross the NixOS/Home Manager boundary

`flake.nix` composes those layers through descriptor attrsets for users, hosts, home profiles, system profiles, and home overlay groups. Three builder functions — `mkOutput`, `mkDarwinOutput`, and `mkHomeOutput` — turn those descriptors into the final flake outputs.
